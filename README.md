## ✨ 특징

- 🎨 **YAML 기반 콘텐츠 팩** - 설정 파일로 쉽게 콘텐츠 정의
- 🧩 **모듈형 기능 시스템** - 유연하고 재사용 가능한 기능 구조
- ⚡ **Paper & Folia 지원** - 최신 서버 환경 호환
- 🔄 **핫 리로드** - 서버 재시작 없이 콘텐츠 갱신
- 📦 **확장 가능한 레지스트리** - 새로운 콘텐츠 타입 추가 가능
- 🎨 **자동 리소스 팩 생성** - Creative API로 손쉽운 팩 생성

## 💻 개발

### 요구사항

- **Java 21**
- **Gradle 8.x**

### 주요 의존성

- [Paper API](https://papermc.io/) - Minecraft 플러그인 API
- [Cloud](https://cloud.incendo.org/) - 명령어 프레임워크
- [Creative](https://github.com/unnamed/creative) - 리소스 팩 API
- Lombok - 코드 간소화
